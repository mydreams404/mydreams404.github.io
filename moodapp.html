MoodApp (ongoing)

Where it started...

the purpose...

To track patterns in daily mood and presenting them in an aesthetically-pleasing way. This idea was born from my personal desire to create beauty out of human emotional experiences, whether they be positive or negative.


the solution...

The solution will be a mobile application closely resembling a calendar. Once a day, the user can set a colour for the day that they feel most accurately represents their overall emotion. Once a set number of days have been logged, the system will generate an image or animation using the colours inputted by the user.

    Here is a list of the features agreed on so far:
  • Can switch between ‘colour selection only’ or ‘notes mode’
  • Can view mood over the course of use and compare
  • Widget to input mood with one touch
  • Notification to remind you to input mood daily (optional)
  • Rearranges mood data to make it look cool/pretty/nice
  • Variety of colour palettes to suit your taste
  • Presents mood data as a wallpaper format
  • User dictates what moods the colour represents
  • Animations are abstract so user can interpret them for themselves
  • Reveal art to user based on number of inputs in the application that is then saved in a gallery for later viewing
  • Art/animations will be written in P5.js
